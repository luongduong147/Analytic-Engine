# Skill: Business Data Analysis

## Overview
This skill provides structured guidance for analyzing business data using the Analytic Engine.

## When to Use
- When user asks to analyze business metrics (revenue, profit, growth)
- When analyzing sales data, customer behavior, or market trends
- When building business dashboards or reports

## Analysis Framework

### Step 1: Define Objective
- Identify the business question
- Determine key metrics to analyze
- Set success criteria

### Step 2: Data Collection
- Fetch relevant data sources
- Validate data quality
- Handle missing values

### Step 3: Exploratory Analysis
- Descriptive statistics
- Trend analysis
- Correlation matrix

### Step 4: Visualization
- Select appropriate chart types:
  - Time series: line chart
  - Comparison: bar chart
  - Distribution: histogram, box plot
  - Proportions: pie chart

### Step 5: Insights & Recommendations
- Key findings summary
- Actionable recommendations
- Next steps

## Example Prompts
- "Analyze Q4 revenue trends"
- "What are the top performing products?"
- "Show customer segmentation by region"
